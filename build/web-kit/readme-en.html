About this kit

  • This kit enables you to distribute your game for Web browsers.
  • Please upload files to your server by ftp.
  • You can't run game by opening HTML file on a local machine.

Steps

  1. Upload game folder to your Web site.
  2. Access http://your-server/game/ and check if sample game runs.
  3. Generate your own data01.arc (Steps are described in tutorial.)
  4. Replace game/data01.arc to your version and upload it to server.
  5. Access again, and check if game works.
    • Reload page or clear browser cache if you see sample game runs.

Limitation

  • Setting window.white is ignored and set to 0 (black)
  • Fade type mask is ignored and set to normal (alpha blending)

Caution

  • Users have to download whole game data whenever access.
  • Insert click wait by @click at the top of your game script because browsers allow sound output after first click.
  • For iPhone/iPad/Android, right click is mapped to two finger tap.
