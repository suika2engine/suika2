Suika2 Webブラウザ版 配布キット

Rev.77 (Suika2/12.22対応)

Copyright (c) 2023 the Suika2 Development Team. All rights reserved.

このキットについて

  • Suika2で作ったゲームをWebブラウザで提供できるようにします
  • ftp等でサーバにアップロードして使います
  • ローカルでファイルを開いても実行できません

手順

  1. gameフォルダをWebサイトにアップロードします
  2. http://サーバ/game/にアクセスしてサンプルゲームが動いているのを確認します
  3. 自分で作ったゲームのdata01.arcを用意します(やり方はチュートリアルにあります)
  4. game/data01.arcを自分で作ったゲームのものに置き換えてアップロードします
  5. 再度アクセスしてゲームが動くか確認します
    • なぜかサンプルゲームが動いてしまう場合、リロードするか、ブラウザのキャッシュをクリアします
  6. ブラウザのタブ名をSuika2から変更する場合、index.htmlを編集してSuika2という箇所を探し、好きな名前に変更してください。

注意

  • ブラウザでアクセス時にゲームデータを一括でダウンロードします
  • ブラウザによっては最初のクリックがあるまで音が出ませんので、クリック待ち(@click)を入れてください
  • iPhone/iPad/Androidで右クリックするには、二本指タップしてください
