    Linux/Chromebook

    From the source code on Ubuntu/Debian/Chromebook (recommended):

    sudo apt-get install -y git build-essential libasound2-dev libx11-dev mesa-common-dev qt6-base-dev qt6-multimedia-dev libwebp-dev
    git clone https://github.com/suika2engine/suika2.git
    cd suika2
    make
    sudo make install
    suika2

    Ubuntu package:

    sudo add-apt-repository ppa:ktabata/ppa
    sudo apt update
    sudo apt install suika2
    suika2

    System Requirements

    Windows

    • Suika2 Pro Desktop (Windows)
      • OS: Windows 11, Windows 10(32-bit/64-bit)
      • CPU: Intel/AMD/Arm64
      • GPU: Direct3D 9
    • Games made with Suika2 (Windows)
      • OS: Windows 11, Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Vista, Windows XP (32-bit/64-bit)
      • CPU: Intel/AMD/Arm64
      • GPU: Direct3D 9

      • macOS 11 Big Sur
      • macOS 12 Monterey
