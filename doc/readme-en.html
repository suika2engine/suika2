About Suika2

Suika2 is an open-source, cross-platform visual novel development engine that allows you to create a feature-rich visual novel with the care and attention to detail that your stories deserve. Featuring support and documentation in Japanese, Chinese, and English as well as active development and an accepting community, Suika2 could be the perfect start to your thrilling adventure, fawning romance, chilling epic, or anything else you could possibly imagine!

Benefits

  • Simple. Easily create robust scripts with Suika2Script.
  • Fast. It's a native application.
  • Free. It's an open source project hosted on GitHub.
  • Diverse. You can develop games on Windows, Mac, Linux, and more.

System Requirements

  • Windows
    • Intel Core 2 Duo or later / Arm 64bit
    • Windows XP, Vista, 7, 8, 8.1, 10, and 11
  • Mac
    • Mac with Apple Silicon or Intel Core
    • macOS 10.13 High Sierra

Usage

  • On Windows:
    • For Playing and Testing:
      • Download the zip file from GitHub and extract it.*
      • Open the suika2 folder in Explorer.
      • Open the suika.exe application.
    • For Development and Debugging:
      • Open the suika2 folder in Explorer.
      • Open the suika-pro.exe application.
  • On Mac:
    • For Playing and Testing:
      • Download the zip file from GitHub and extract it.*
      • Open the suika2 folder in Finder.
      • Copy the suika application in mac.dmg to the suika2 folder.
      • Open the suika application in the suika2 folder.
    • For Development and Debugging:
      • Open the suika2 folder in Finder.
      • Copy the Suika2 Pro for Creators application in mac-pro.dmg to the suika2 folder.
      • Open the Suika2 Pro for Creators application in the suika2 folder.
  • On iPhone/iPad:
    • Build from the source code using Xcode. See build/README.md or visit the Wiki.
  • On Android:
    • A sample game is available on Google Play. See the Google Play Store.
    • To make your own game, you have to build from the source code using Android Studio. See build/README.md or visit the Wiki.
  • On Linux/FreeBSD/NetBSD:
    • Build from the source code. See build/README.md or visit the Wiki.
